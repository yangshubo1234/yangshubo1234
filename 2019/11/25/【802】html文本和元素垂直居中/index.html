元素的垂直居中

1，使用absolute与transform配合实现

步骤：

  • 首先给父级元素添加相对定位position: relative;
  • 然后给子级元素添加绝对定位 position: absolute;（即需要垂直居中的元素）
  • 最后让子级距离父级左边和上边分别为left: 50%; top: 50%; transform: translate(-50%, -50%);

display flex 弹性容器

align-items center 定义项目在交叉轴（纵轴）上如何对齐，垂直对齐居中
