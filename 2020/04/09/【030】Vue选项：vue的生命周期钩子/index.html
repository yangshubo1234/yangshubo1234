什么是生命周期

vue 的每个组件就像一个生命体，有出生，有成长，有死亡等阶段，它们会在不同的阶段做不同的事，这个过程就是生命周期。

vue组件生命周期有以下几个：

周期 说明
beforeCreate 创建前
created 创建后
beforeMount 挂载前
mounted 挂载后
beforeUpdate 更新前
updated 更新后
beforeDestroy 销毁前
destroyed 销毁后
