发表于 2019-11-08 | 分类于 javascript

参考文档：Object.defineProperty

一，语法

参数 说明
obj 需要定义属性的当前对象
prop 当前需要定义的属性名
desc 属性描述符

二，示例

参数 说明
value 属性的初始化值
writeble 是否可修改值的内容
enumerable 是否可枚举，默认为false
configurable 是否可删除，默认为false
get 在读取属性时调用的函数，默认值为undefined
set 在写入属性时调用的函数，默认值为undefined
